MumbleWumble is a simple way to encrypt and decrypt secret messages using our own encryption algorithm. It transforms readable text into a scrambled format that can only be reversed with the same system, making it useful for learning, experimentation, and sending hidden messages.

***Features***

- Secret Codeword When Encrypting To Ensure Maximum Security
- Simple Interface
- Cool Color Scheme

<small>Note: DO NOT use this in an professional setting</small>
